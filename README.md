## Basic Information

* Compiler: XeLaTeX
* Fonts: Victor Mono (ligature support), FontAwesome (icons)
* Packages:
	* xltxtra (for fontspec)
	* geometry (margins)
	* fontawesome (from [posquit0/latex-fontawesome](https://github.com/posquit0/latex-fontawesome))
